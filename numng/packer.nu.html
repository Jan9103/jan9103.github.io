Numng packer.nu compatability

Numng supports big parts of the packer.nu package standard.

Using basic packer.nu packages

Most packages should just work out of the box if you mark them as packer.nu packages by adding 

"package_format": "packer"
to their definition.

Packer.nu standard library

Packer.nu added a standard library, which was not used often.

If a package requires it you can import 90% of the standard library using the 

{"name": "jan9103/packer.nu"}
package.
