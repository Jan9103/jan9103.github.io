Installation

  1. Open a nushell instance
  2. Run 
    http get --raw "https://raw.githubusercontent.com/Jan9103/numng/main/numng.py" | save -r numng.py
    todownload numng, which includes a install-assistant.
  3. Run 
    python3 numng.py --nu-config init
    . This will run the install-assistant and do the following:
    1. It generates a basic numng.json with a dependency on numng itself. By installing numng using itself it can self-update in the future.
    2. It append the loader-snippet to your nushell config.
    3. It executes 
      numng --nu-config build
      in order to apply the generated numng.json and install numng that way.

Post-installation steps

  • You can find your numng.json in a directory called numng within your nushell config directory (on linux: 
    ~/.config/nushell/numng/numng.json
    ).
  • You can find a list of packages at the package index
  • You can find a basic usage guide here
  • Usually numng installed binaries (including itself) are only available within nushell. But you can just add 
    ~/.local/share/nushell/numng/nu_config_nupm_home/bin
    to the PATH to change that.
    • This makes it possible to install nushell via numng: 
      {"name": "nushell/nushell", "version": "latest"}
      (you can also use 
      "version": "git"
      )
